# Hungry Panda game (WIP)

A simple HTML5 game built using phaser.js framework.

### Objective
To collect as many fruit as you (the panda) can before time runs out.  Watch out for bombs!

### Scoring System
10 pts / fruit

-5 pts / bomb hit

### Health System
Level 1:  2 lives
Level 2:  3 lives
Level 3:  4 lives

-1 life / bomb hit
